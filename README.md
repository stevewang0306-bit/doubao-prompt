# 豆包(Doubao)系统提示词

字节跳动豆包(ByteDance Doubao)模型的系统提示词(System Prompt)记录,仅供学习与研究参考。

## 这是什么

系统提示词是 AI 产品在每次对话开始前,预先注入给模型的一段隐藏指令,通常决定了它的身份设定、行为规范、回答风格以及安全边界等。本仓库记录的就是豆包模型的一份系统提示词内容。

- **来源模型**:字节跳动豆包(Doubao),基于 Seed 大模型基座
- **记录时间**:2026 年 5 月
- **提示词原文**:见 [`prompt.md`](./prompt.md)(纯文本)

## 用途说明

本仓库出于对 AI 系统透明度(transparency)的兴趣而整理,方便研究者、开发者以及好奇的人,了解这类产品"幕后"的指令设计思路,**不用于任何商业目的**。

# Doubao system prompt word

The record of system prompts for the ByteDance Doubao model is provided for learning and research reference only.

## What is this

The system prompt is a hidden instruction pre-injected into the model before each conversation starts for AI products, which usually determines its identity setting, behavior norms, response style, and security boundaries. This repository records the content of a system prompt for the Doubao model.

- **Source Model**: ByteDance Doubao, based on the Seed large model foundation
- **Record time**: May 2026
- **Original prompt**: See [`prompt.md`](./prompt.md) (plain text)
